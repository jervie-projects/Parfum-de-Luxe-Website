# Parfum de Luxe Website
Parfum de Luxe is an online store website project developed strictly for school work only, it is designed to run locally on XAMPP.

Some images used in this project were sourced online for educational purposes only.  
They are included here solely for demonstration and learning, and are **not intended for commercial use**.  
All rights for these images belong to their respective owners.

## Project Features
- User authentication (login/logout)
- Database integration with MySQL
- PHP-driven dynamic pages
- Basic front-end using HTML, CSS, and JavaScript

## How to Run Locally
1. Download this repository and unzip the folder.
2. Move the project folder to your XAMPP `htdocs` directory.
3. Import the database file named "db_perfume-season" into phpMyAdmin.
4. Start Apache and MySQL from the XAMPP Control Panel.
5. Open your browser and navigate to `http://localhost/3B-11`.

## Technologies Used
- PHP
- MySQL
- XAMPP
- HTML, CSS, JavaScript
